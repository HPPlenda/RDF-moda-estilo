# RDM-moda-estilo
RDM Moda &amp; Estilo é um e-commerce moderno e responsivo focado em vestuário masculino, feminino e infantil. Desenvolvido originalmente em React e convertido para HTML, CSS e JS puro, o projeto conta com navegação fluida, sistema de busca em tempo real, abas de categorias e um carrinho de compras intuitivo. Essencial para o novo varejo digital agora!
